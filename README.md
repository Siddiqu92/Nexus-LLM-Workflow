# Nexus-LLM-Workflow: AI-Powered RAG Platform

Production-grade LLM application built with Next.js 14, LangChain, and Pinecone. This platform automates complex AI workflows with Retrieval-Augmented Generation (RAG).

![AI Conversation Preview](/public/images/chat-conversation.png)

## Key Features
- Advanced RAG Pipeline: Context-aware responses using Pinecone Vector DB for high-speed retrieval.
- Cost Efficiency: Smart model routing between OpenAI & Anthropic Claude, reducing API costs by 30%.
- Structured Output: Precise data extraction and validation using Zod schema.
- Agentic Workflows: Multi-step complex tasks managed via LangGraph.js.
- Real-time Streaming: Smooth UI experience with real-time token streaming via Vercel AI SDK.

![Structured Output Preview](/public/images/structured-output-conversation.png)

## Tech Stack
- Framework: Next.js (App Router), TypeScript
- AI Orchestration: LangChain.js & LangGraph.js
- Vector Store: Pinecone / Supabase
- Styling: Tailwind CSS & Shadcn UI
